Principais formatações:

Negrito / Destaque

Nesta frase temos um termo em destague podendo ser de semântico (usando a tag b) ou Não semântica (usando a tag strong)

Itálico / Ênfase

Nesta frase,temos um termo com enfase semântico (usando a tag em) e Não semântico (usando a tag I).

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto Grande ou Pequenos

Estamos criando um Texto grande e um Texto pequeno.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido,mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar Ênfase.(semântico)
Existe também com a Tag U ,para marcar o texto (Não semântico)
